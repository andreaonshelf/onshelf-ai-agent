Check Insert Buttons in Dashboard

Open this page, then navigate to http://localhost:8130 in another tab.

Steps to Check:

  1. Open the dashboard at http://localhost:8130
  2. Click on "Extraction Config" tab
  3. Open browser console (F12)
  4. Run these commands:
// 1. Check if FieldEditor component exists
console.log('FieldEditor exists:', typeof FieldEditor !== 'undefined');

// 2. Check if insert buttons are in DOM
const insertButtons = document.querySelectorAll('.insert-field-button');
console.log('Insert buttons found:', insertButtons.length);

// 3. Check insert wrappers
const insertWrappers = document.querySelectorAll('.insert-field-wrapper');
console.log('Insert wrappers found:', insertWrappers.length);

// 4. Make all insert buttons visible
document.querySelectorAll('.insert-field-wrapper').forEach(el => {
    el.style.opacity = '1';
    el.style.background = 'yellow';
    el.style.padding = '10px';
    el.style.border = '2px solid red';
});

// 5. Check if field editor container exists
const fieldEditor = document.querySelector('.field-editor-container');
console.log('Field editor container:', fieldEditor);

// 6. Check all buttons in the extraction config area
const allButtons = document.querySelectorAll('.stage-content button');
console.log('All buttons in stage content:', allButtons.length);
allButtons.forEach(btn => console.log('Button text:', btn.textContent));

// 7. Check if fields are being rendered
const fieldItems = document.querySelectorAll('.field-item');
console.log('Field items found:', fieldItems.length);

// 8. Force create an insert button
if (fieldEditor && insertButtons.length === 0) {
    const testButton = document.createElement('div');
    testButton.innerHTML = '';
    fieldEditor.insertBefore(testButton, fieldEditor.firstChild);
    console.log('Test button added!');
}

If no insert buttons found:

The issue might be:

  • The onInsertFieldAt prop is not being passed
  • The FieldEditor component is not rendering them
  • CSS is hiding them
  • JavaScript error preventing render
