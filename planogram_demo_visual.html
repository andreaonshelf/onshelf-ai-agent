üîç Visualization Concepts Explained

üìê Concept 1: Each Facing Gets Its Own Cell
Notice how Coca-Cola Zero with 3 facings occupies 3 separate grid cells (F1, F2, F3), not one cell labeled "3√ó1".
üìö Concept 2: Stacking Visualization
Powerade and Gatorade on Shelf 3 show 2√ó2 stacking. Each product has 2 facings (width) √ó 2 stack (depth), creating 4 total units in a 2√ó2 grid arrangement.
üåê Concept 3: Global Width Consistency
All shelves have exactly 8 grid columns, even if they don't use all slots. This ensures visual alignment across all shelf levels.
üëª Concept 4: Transparent Empty Slots
Empty slots are completely transparent with just a dashed border and mailbox icon. They take exactly 1 grid position.

üéØ Key Takeaways for LLM Extraction

‚úÖ DO:

  • Count exact facings: "columns": 3
  • Specify stack depth: "stack": 2
  • Include empty slots in sequence
  • Process Left ‚Üí Center ‚Üí Right
  • Use sequential position numbers

‚ùå DON'T:

  • Use vague terms: "multiple facings"
  • Combine facings: "3√ó1" in one cell
  • Skip empty positions
  • Use grid position as JSON position
  • Make shelves different widths
